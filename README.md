# 🤖 AI Cyber Toolkit

Expérimentations **AI + cybersécurité** pour labs SOC : détection d'anomalies simple, helpers Python.

## Contenu

- `anomaly_detector.py` — baseline statistique / score d'anomalie
- `requirements.txt`

## Quick start

```bash
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt
python anomaly_detector.py --help
```

## Scope

Outil **éducatif** — pas un EDR de production. Idéal pour comprendre ML basique appliqué aux logs/metrics.
